ACTIVIDAD FINAL

Este layout fue construido usando únicamente Flexbox y las propiedades vistas en clase: justify-content, align-items, flex, flex-direction y gap. 

nav      → display:flex · justify-content:space-between · align-items:center
main     → display:flex · flex-direction:row (por defecto)
aside    → flex:1
article  → flex:3
footer   → display:flex · 3 sections con flex:1 cada una

Propiedades usadas

justify-content align-items flex-direction flex: 1 / 3 gap flex-wrap

Actividad

Layout completo usando solo propiedades Flexbox vistas en clase.

Técnica

Flexbox anidado: nav, main y footer son contenedores flex independientes.

Nota

Sin Grid, sin float, sin position absolute. Solo display: flex.
