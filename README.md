# ISW-521 - Programación en Ambiente Web I

## Descripción

Este repositorio contiene las prácticas, laboratorios, ejercicios y actividades desarrolladas durante el curso **Programación en Ambiente Web I (ISW-521)** de la **Universidad Técnica Nacional (UTN)**.

El objetivo de este repositorio es documentar el proceso de aprendizaje y servir como respaldo del trabajo realizado durante el curso, incluyendo ejemplos de código, ejercicios prácticos y proyectos relacionados con el desarrollo de aplicaciones web.

## Temas abordados

Entre los principales temas estudiados se encuentran:

* Fundamentos de Internet y la Web.
* Protocolos de comunicación (HTTP, HTTPS, TCP/IP).
* HTML5 y estructura de páginas web.
* CSS3 y diseño responsivo.
* Flexbox y CSS Grid.
* JavaScript y programación del lado del cliente.
* Manipulación del DOM y eventos.
* Almacenamiento en el navegador.
* Git y GitHub para control de versiones.
* Frameworks y librerías para desarrollo web.
* Consumo de APIs y servicios web.
* Buenas prácticas de desarrollo y seguridad web.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
* Node.js
* Frameworks y librerías vistas en el curso

## Objetivo académico

Desarrollar habilidades en el diseño, construcción y mantenimiento de aplicaciones web modernas, aplicando estándares de la industria, buenas prácticas de programación y herramientas de desarrollo profesional.
